# 照骨灯

照骨灯是用于检测谐振腔镀层内部缺陷、晦暗和微观裂纹的专用检测工具。

## Scope
适用于封窖前检查、深度清理前全量扫描、坍缩后排查和换羽决策。照骨灯的作用不是检测散斑本身，而是识别会持续诱发散斑和结构失效的镀层问题；当反射率或完整性不达标时，通常需要安排换羽或停止封存计划。
