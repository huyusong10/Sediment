# 溢彩

溢彩是谐振腔镀层完好且防护性能处于理想状态时的外观与运行表征。

## Scope
适用于镀层巡检、换羽后验证和运行质量判断。材料表明溢彩阶段通常伴随稳定嗡鸣度、较低底噪、较小峰谷差、较高清浊比和极低散斑增长率，因此它可作为镀层健康与恢复完成的判据；换羽后若未恢复到溢彩，不应视为闭环完成。
