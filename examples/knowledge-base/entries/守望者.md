# 守望者

守望者是负责监控静默态和值班安全的核心角色，其职责是确保系统在非工作时段仍处于受控、可交接和可应急的状态。

## Scope
适用于全天候值班、晨祷晚课、交接班和异常上报流程。守望者需要持续监测嗡鸣度、盲区、温存参数和异常事件，并在红线、断流、回流或幽灵读数等场景下按既定升级路径通知相关角色，因此该角色既承担监控义务，也承担事件流转责任。
