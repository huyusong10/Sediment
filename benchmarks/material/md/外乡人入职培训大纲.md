# 外乡人入职培训大纲

## 培训基本信息

| 字段 | 内容 |
|------|------|
| 适用对象 | 外乡人（入职 0-90 天的新员工） |
| 培训周期 | 30 个工作日 |
| 导师配置 | 每位外乡人配备一名老把式作为专属导师 |
| 考核方式 | 笔试 + 实操 + 导师评价 |

## 1. 哈基米基础理论（第 1-3 天）

### 1.1 哈基米的本质

**哈基米**是本行业一切活动的能量基础，也是系统的核心资源与物质基础。所有工程师必须理解哈基米的以下特性：

- 哈基米以离散态存在，具有可量化的嗡鸣度
- 哈基米可以在谐振腔之间传输，遵循旋涡协议
- 哈基米可以被晶格化为各种可用形态
- 哈基米泄漏会产生散斑，需要清道夫流程清理

### 1.2 谐振腔原理

**谐振腔**是存储和处理哈基米的核心设备。培训内容：

- 谐振腔的基本结构与工作原理
- 谐振腔的嗡鸣度调谐方法（定音鼓操作）
- 谐振腔容量与饱和度的概念
- 谐振腔镀层的作用与维护周期

### 1.3 实操环节

- 在导师指导下完成一次谐振腔试音
- 使用量天尺测量基础传输损耗
- 识别回音壁上的嗡鸣度读数

## 2. 日常运维流程（第 4-7 天）

### 2.1 晨祷

**晨祷**是每日系统启动的标准检查流程，所有工程师必须掌握：

1. 检查所有谐振腔状态指示灯
2. 确认回音壁嗡鸣度读数在共振峰区间内
3. 验证千机匣自动化控制链路正常
4. 检查留声机存储空间和数据完整性
5. 执行对钟操作，校准时间同步
6. 签署晨祷确认单

### 2.2 晚课

**晚课**是每日关闭前的数据归档流程：

1. 汇总当日嗡鸣度数据并归档至留声机
2. 记录当日哈基米收支情况，提交账房系统
3. 检查是否有待处理的散斑清理任务
4. 确认次日值班安排和守望者交接事项
5. 签署晚课确认单

### 2.3 实操环节

- 跟随导师完整参与三次晨祷和晚课
- 独立完成一次晨祷检查清单的填写
- 学习使用账房系统录入基础数据（演练环境）

## 3. 安全规范与权限体系（第 8-12 天）

### 3.1 权限等级

哈基米系统的权限体系如下：

| 权限等级 | 角色 | 核心职责 |
|---------|------|---------|
| L1 | 外乡人 | 只读观察，生产操作必须由导师代执行 |
| L2 | 正式工程师 | 独立操作谐振腔和传输流程 |
| L3 | 老把式 | 指导外乡人、审批变更、处置异常 |
| L4 | 祭司团 | 制定规范、协议修订、架构决策 |
| L5 | 掌灯人 | 最高权限，负责启明仪式和重大决策 |

**掌灯人**是系统中权限最高的角色，也是负责启明仪式的最高权限管理员。外乡人在培训期间不得直接与掌灯人对接工作，所有涉及 L5 权限的事项需通过导师转达。

外乡人的权限限制如下：

- 仅可查看嗡鸣度等基础监控数据，属于只读权限
- 不能执行启明、泄洪、织网、锁龙井等生产操作
- 不能修改系统配置或审批任何变更
- 所有实操训练仅限导师陪同下的演练环境，不得直接作用于生产腔体

### 3.2 启明仪式

**启明**是新谐振腔在完成开光与试音后的首次受控放量仪式，具有特殊的行业意义。启明必须由掌灯人亲自执行或授权执行。外乡人可以作为观察员参与启明仪式，但不得操作任何设备。

启明流程要点：

1. 掌灯人确认谐振腔完成开光和试音
2. 执行对钟确保时序同步
3. 受控注入初始哈基米，正式注入量不超过额定容量的 10%，嗡鸣度逐步提升至共振峰
4. 晶格化为初始形态，注入点睛参数
5. 进入试运行并完成稳定性验证后，方可切换至正式运行状态

### 3.3 红线与三振法则

**红线**是嗡鸣度绝对不可超过的安全阈值。任何情况下，嗡鸣度接近红线时都必须立即采取降频措施。触碰红线将触发看门狗自动记录。

**三振法则**是系统的核心安全策略：

- 连续三次出现同类异常（如嗡鸣度超限、断流、违规操作），相关腔体或操作者将被自动隔离
- 隔离期间需由老把式级别的工程师进行根因分析
- 三次隔离记录将触发看门狗的强制停机程序

### 3.4 值班体系

| 角色 | 职责 | 值班时段 |
|------|------|---------|
| 守望者 | 监控静默态，确保无异常哈基米流动 | 全天轮班 |
| 守夜人 | 夜间系统巡检、异常初步处置 | 22:00 - 06:00 |

外乡人在培训期间将安排一次跟随守夜人的夜间值班体验，了解夜间运维的特殊要求。

## 4. 冷启动与应急响应（第 13-15 天）

### 4.1 冷启动流程

**冷启动**指系统从完全关闭状态恢复到可用状态的全过程。冷启动是外乡人必须掌握但不得独立执行的操作（需 L2 及以上权限）。

冷启动关键步骤：

1. 硬件自检：确认所有谐振腔、回音壁、千机匣硬件正常
2. 晨祷初始化：执行完整的晨祷检查清单
3. 对钟同步：校准所有腔体时钟
4. 嗡鸣度预热：逐步提升嗡鸣度至底噪水平
5. 安全弦校验：确认运行参数在安全边界内
6. 信使链路验证：测试状态传递通道
7. 试运行：执行一次小额哈基米传输测试

### 4.2 应急响应

外乡人需要了解的应急响应知识：

- 发现红线告警时立即通知导师和守望者
- 发现散斑泄漏时启动清道夫流程并上报园丁团队
- 发现可疑暗流时保护现场并通知渡鸦团队
- 不得擅自操作锁龙井、判官等安全设备

## 5. 行业文化与职业发展（第 16-20 天）

### 5.1 行业角色认知

了解行业中的主要角色和团队：

- **祭司团**：制定晨祷、晚课等日常规范，维护旋涡协议
- **园丁团队**：负责散斑清理和设备日常维护
- **听风者**：分析嗡鸣度模式，提供数据洞察
- **渡鸦**：暗流调查与安全防护
- **调音师**：谐振腔嗡鸣度调谐专家

### 5.2 导师制度

每位外乡人的导师由经验丰富的老把式担任。老把式拥有十年以上行业经验，负责：

- 指导外乡人掌握日常操作流程
- 解答技术疑问
- 评估外乡人的实操能力
- 在 L30 天培训结束时出具评价报告

### 5.3 培训考核

培训结束时需通过三项考核：

1. **笔试**：哈基米基础理论、安全规范、流程知识（及格线 80 分）
2. **实操**：在导师监督下于演练环境完成一次晨祷、一次传输操作和一次清道夫流程
3. **导师评价**：老把式对外乡人的学习态度、操作规范性和安全意识进行综合评分

考核通过者将获得 L2 权限，正式成为哈基米行业的一员。
