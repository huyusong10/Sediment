# 祭司团

祭司团是制定系统架构原则、标准流程和关键审批边界的规则制定主体。

## Scope
适用于架构评审、晨祷晚课规范、行业级对钟校准和隔离解除授权等治理场景。材料显示祭司团既维护行业术语和流程，也对重大变更与关键解除动作保留审批权，因此它代表的是制度控制面，而不是单一执行团队。
