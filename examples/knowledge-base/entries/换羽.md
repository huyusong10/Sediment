# 换羽

换羽是对谐振腔镀层进行定期或异常触发更换的维护操作。

## Scope
适用于镀层老化、晦暗出现、溢彩消失或事故后受损的谐振腔。换羽前需先通过温存完成卸载并在静默态作业，不能带载执行；换羽结束后通常需要重新验证运行状态，因此它主要解决镀层健康问题而不是结构性故障。
