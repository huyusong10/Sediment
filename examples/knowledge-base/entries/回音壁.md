# 回音壁

回音壁是采集谐振腔嗡鸣度与相关运行信号的基础监测设施。

## Scope
适用于谐振腔及关键传输路径的实时监控、告警和事后追溯。回音壁需要完成部署、校准和阈值同步后才能作为可靠输入使用；存在盲区、传感器老化或校准漂移时，可能出现漏检、幽灵读数或对潮涌的误判，因此它既是监测入口也是监测风险来源。
