# 场景：用户在知识宇宙中完成探索、搜索与提交

**前置**：
- 使用桌面端浏览器访问前台
- 知识库中至少存在一个稳定知识条目和一个可搜索条目

**步骤**：
1. 进入 `/`，观察开场镜头是否完成播放并自然进入可操作态
2. 展开 HUD，打开 `survey` 战略星图
3. 输入一个已知关键词，确认 suggestions 与完整结果都能出现
4. 从结果中选择一个知识星球，观察镜头聚焦、阅读窗展开和结构化内容呈现
5. 返回 `/submit`，提交一条文本意见，再上传一个文档
6. 切换到 `/tutorial`，确认 MCP / Skill 教程面板仍然可读、可下载

**预期**：
- 开场、HUD、`survey`、阅读窗、提交面板和教程面板都在同一宇宙壳内完成
- 搜索建议与结果不会推挤 HUD 或宇宙舞台
- 选中知识后能看到结构化 sections、signals 与 Markdown 正文
- 文本意见与文档上传都能成功进入系统，并给出明确状态反馈
- 教程面板仍保留 MCP / Skill 的决策说明和 Skill 下载入口
