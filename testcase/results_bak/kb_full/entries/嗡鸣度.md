# 嗡鸣度

嗡鸣度是衡量谐振腔中哈基米活跃程度的指标，单位为 Hz，与饱和度和能量活动强度正相关。

## 上下文

标准谐振腔嗡鸣度参数：
- 红线（绝对上限）：720.0 Hz
- 大型谐振腔红线：680.0 Hz
- 共振峰（理想运行区间）：420.0 - 580.0 Hz
- 底噪（系统固有最低嗡鸣度）：100.0 Hz
- 采样频率：100 Hz
- 回音壁刷新间隔：30 秒

嗡鸣度与饱和度的关系：base + (saturation / 0.85) * (red_line - base)。当嗡鸣度超过红线时触发三振法则计数。
