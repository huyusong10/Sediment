# 旁路画像核对映射

## Hypothesis
当前安全调查条目已经默认存在一套“回音壁采样 -> 照妖镜快照 -> 溯光来源”的核对顺序，只是还没有把这套映射工件沉淀出来。

## Proposed Answer
旁路画像不应只是一张截图，而应是一张对照映射：用回音壁样本提供基线，用照妖镜补足被隐身衣遮蔽的传输快照，再用溯光确认来源链路和时间段。只有三类证据能相互指向同一条异常路径时，才应把现象判定为真实旁路，而不是幽灵读数或偶发噪声。
