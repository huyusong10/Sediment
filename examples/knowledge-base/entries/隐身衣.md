# 隐身衣

隐身衣是通过反相嗡鸣度场抵消传输信号、让暗流在常规监测中近乎不可见的规避检测手段。

## Scope
适用于解释回音壁出现异常空白段、暗流路径难以被常规监测捕获、以及账房异常与现场监测不一致的安全调查场景。隐身衣能绕过常规监测，但无法消除物理库存变化和账房收支差额，因此它提高的是发现难度，不是使暗流不可被追查。
