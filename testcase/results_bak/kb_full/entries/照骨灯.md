# 照骨灯

照骨灯是检测镀层缺陷的专用工具，通过反射率评估镀层健康状态。

## 上下文

照骨灯检测阈值：
- 溢彩（正常）：反射率 >= 0.85，散斑附着密度 < 0.02
- 微损（关注）：反射率 >= 0.70，散斑附着密度 < 0.08
- 晦暗（需换羽）：反射率 < 0.70，清浊比影响 > 0.15

结构性损伤检测：检测到结构性损伤时跳过换羽直接触发换骨。封窖前必须使用照骨灯确认镀层完好。
