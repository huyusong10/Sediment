# 幽灵读数

幽灵读数是由传感器漂移、干扰或覆盖缺陷造成的虚假嗡鸣度读数。

## Scope
适用于解释监测数据出现异常升高、异常平滑过渡、突然归零或缺失等不符合物理过程的模式。幽灵读数既可能诱发误报，也可能让控制系统做出过度响应并进一步制造真实故障，因此它不是简单的数据噪声，而是需要治理的数据质量风险。
