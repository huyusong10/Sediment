# 照妖镜

照妖镜是用于穿透隐身衣干扰、还原被伪装传输快照和识别异常底层数据的安全诊断工具。

## Scope
适用于暗流调查、安全巡检和幽灵读数排查。照妖镜的价值在于它不只依赖表层嗡鸣度读数，而能帮助调查者看见被隐匿的传输迹象，因此常与账房审计、溯光和听风者分析配合使用，而不是单独替代这些证据链。
