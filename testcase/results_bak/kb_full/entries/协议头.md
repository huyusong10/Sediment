# 协议头

协议头是旋涡协议消息的前 16 字节元数据。

## 上下文

协议头字段：
- version（1 字节）：协议版本号
- msg_type（1 字节）：消息类型
- sequence（2 字节）：序列号
- reserved（4 字节）：保留字段
- source（16 字节）：源节点 ID
- destination（16 字节）：目标节点 ID
- payload_length（2 字节）：载荷长度
