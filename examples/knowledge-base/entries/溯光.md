# 溯光

溯光是通过嗡鸣度指纹或残留特征回溯哈基米来源路径、泄漏时间和源腔归属的追踪技术。

## Scope
适用于散斑溯源、暗流调查和异常批次来源确认。溯光不依赖单一运行日志，而是利用样本和历史特征重建来源，因此在日志被篡改、绕过旋涡协议或污染已扩散后仍具有调查价值；其结果常被用于制定换羽计划、安全处置和责任确认。
