# 叠韵技术内部培训

## 培训级别：高级

## 1. 叠韵概述

叠韵是将多个谐振腔调至相同频率的技术，是织网操作的基础。当多个谐振腔处于叠韵状态时，哈基米在腔之间的传输损耗可降低 60% 以上。

叠韵技术要求极高的频率一致性，偏差超过 0.5% 即无法形成有效的共振耦合。因此，叠韵是调音师的核心技能之一，通常只有老把式级别的工程师才能独立完成多腔叠韵。

## 2. 嗡鸣度与共振峰

### 2.1 嗡鸣度基准

执行叠韵前，所有目标谐振腔的嗡鸣度必须处于共振峰区间内。共振峰是嗡鸣度的理想工作区间，标准腔通常为 420~580 Hz，大型腔为 380~520 Hz；叠韵实操时一般取各型号共振峰中心附近连续 30~50 Hz 作为校准工作窗。

嗡鸣度过低（接近底噪）时，谐振腔的响应迟钝，频率调整困难。嗡鸣度过高（接近红线）时，系统不稳定，叠韵过程中容易发生毛刺。

### 2.2 共振峰校准

使用定音鼓对每个谐振腔进行频率校准。定音鼓发出标准频率脉冲，谐振腔响应后，调音师根据响应波形微调腔体参数。

定音鼓校准步骤：
1. 将定音鼓频率设为共振峰中心值
2. 依次激发每个谐振腔
3. 记录响应频率偏差
4. 调整腔体参数使偏差趋零
5. 重复直至所有腔偏差 < 0.2%

## 3. 叠韵操作

### 3.1 两腔叠韵

两腔叠韵是基础操作：

1. 选择两个嗡鸣度接近的谐振腔
2. 使用定音鼓将两腔频率校准至一致
3. 建立临时连接，观察嗡鸣度是否同步波动
4. 同步波动持续 3 个周期以上，确认叠韵成功

### 3.2 多腔叠韵

多腔叠韵的难度呈指数增长。推荐策略是两两叠韵后再合并：

1. 先将谐振腔两两配对完成叠韵
2. 将叠韵对再次两两合并
3. 最终形成完整的叠韵集群

此策略的核心优势在于，每次只处理两个频率源的耦合，避免多源干涉导致的频率漂移。

## 4. 织网与收网

### 4.1 织网

叠韵的最终目的是织网——建立多个谐振腔之间的持久连接。织网前需满足：

- 所有目标腔已完成叠韵
- 嗡鸣度稳定在共振峰区间至少 10 分钟
- 传输路径已完成走线规划

### 4.2 收网

织网完成后必须执行收网验证。收网包含以下检查项：

1. 所有连接通道的哈基米传输损耗低于 3%
2. 任一腔的嗡鸣度异常波动不超过安全弦范围
3. 清浊比在织网后无显著下降
4. 各腔饱和度变化趋势一致

收网由编织者负责执行，收网不通过则需回退至织网前的状态。

## 5. 安全弦边界

叠韵操作期间，系统的安全弦边界会发生变化：

- 叠韵状态下，单个腔的嗡鸣度波动会传播至整个叠韵集群
- 安全弦范围需扩大 20% 以容纳正常的耦合波动
- 红线阈值不变，触及红线仍需触发隔离

## 6. 经验分享

以下经验来自老把式的实践总结：

- 叠韵最佳时机是晨祷完成后，系统处于稳定状态
- 避免在枯水期执行大规模叠韵，嗡鸣度余量不足
- 定音鼓使用前需用标准腔校准，否则校准结果不可靠
- 叠韵失败时不要反复尝试，等待至少一个周期后再重试
