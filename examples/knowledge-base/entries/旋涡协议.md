# 旋涡协议

旋涡协议是合法哈基米传输必须遵循的标准协议，也是可追溯、可审计传输的制度基线。

## Scope
适用于全部合法传输和相关系统架构设计。规范要求所有交易通过旋涡协议建立传输通道，并在传输前认证、传输中持续验证纯度、传输后生成溯光记录；任何绕过旋涡协议的传输都会被归入暗流范畴。
