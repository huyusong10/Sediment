# 晶格化

晶格化是将离散哈基米通过算法固化为稳定形态的处理过程。

## 上下文

晶格化参数：
- 最低清浊比要求：9.0
- 晶格尺寸：8x8 矩阵
- 标准镀层厚度：1.0
- 晶格化损耗：5%（volume * 0.95）
- 晶格化后清浊比提升：约 1.5 倍

晶格等级评定：
- PERFECT：清浊比 >= 49.0 且均匀性 >= 0.95
- PRIME：清浊比 >= 19.0 且均匀性 >= 0.90
- STANDARD：清浊比 >= 9.0
- DEFECTIVE：低于标准
