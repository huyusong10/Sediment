# 断流

断流是哈基米传输过程中流动意外中断的异常，通常需要按持续时间分级响应并快速恢复链路可用性。

## Scope
适用于旋涡协议传输、织网故障排查和晨祷对钟后时序异常治理。断流不是所有传输暂停的统称，而是运行中发生的非预期中断；持续时间越长，处置就越会从自动重试升级到热备切换和源腔锁定。
