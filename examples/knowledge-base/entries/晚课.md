# 晚课

晚课是每日关闭前对运行数据、异常处理和次日准备进行归档收口的标准流程。

## Scope
适用于日终停运、启明后归档和运维交接前的数据闭环场景。晚课不只是简单存档，而是同时要求汇总嗡鸣度历史、核对账房收支、确认待处理任务并生成次日晨祷输入，因此它是治理闭环的一部分，不是可随意省略的收尾动作。
