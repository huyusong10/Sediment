# 风火轮

风火轮是通过预加载参数并缩短连接建立时间来加速跃迁的性能优化技术。

## Scope
适用于高频跃迁路径优化和金蝉脱壳等对时延敏感的迁移场景。材料表明风火轮既可降低跃迁前的建立耗时，也可在故障迁移中缩短总时长，但它并不替代对钟、共振峰对齐或安全弦检查；参数越界时，风火轮只会更快地放大风险。
