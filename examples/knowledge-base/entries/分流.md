# 分流

分流是把单一路径上的消息或状态拆分成多路并行投递，以降低单路拥塞或同时覆盖多个目标的通信手段。

## Scope
适用于多目标通知、负载分散和并行同步场景。材料中的分流作用于传递过程而非原始哈基米本体，因此它描述的是信使层的通信策略，不等同于跃迁或物料迁移。
