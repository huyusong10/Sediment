# 镀层

镀层是谐振腔用于隔绝外部干扰并抑制散斑渗入的防护外壳。

## Scope
适用于谐振腔运行、维护、换羽和老化评估场景。材料表明镀层状态会同时反映在外观、嗡鸣度模式、底噪和清浊比上，并直接影响散斑增长与故障风险；因此镀层不是单纯外壳，而是运行质量和寿命管理的关键约束面。
