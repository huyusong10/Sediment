# 启明仪式操作 Checklist

## 文档编号：QM-CHK-001

## 1. 前置条件

启明是新谐振腔在开光与试音完成后的首次受控放量仪式，属于“开光 -> 试音 -> 启明 -> 试运行 -> 正式运行”链路中的正式投运步骤。启明仪式必须由掌灯人主持或授权执行。

**确认项**：
- [ ] 掌灯人已到场或远程授权
- [ ] 祭司团已审批启明方案
- [ ] 新谐振腔完成全部建造验收
- [ ] 谐振腔周围 50 米内无散斑污染
- [ ] 回音壁已部署并校准完毕

## 2. 开光前检查

开光是新谐振腔首次启用的准备流程。

**检查项**：
- [ ] 谐振腔结构完整性检测通过
- [ ] 镀层检测通过，溢彩光泽均匀无晦暗
- [ ] 饱和度传感器校准完成，读数归零
- [ ] 嗡鸣度传感器校准完成，底噪读数在正常范围内
- [ ] 传输接口密封性测试通过
- [ ] 锁龙井处于待命状态（紧急情况下可立即锁定）

## 3. 试音流程

试音是测试谐振腔是否可用的关键步骤。

**步骤**：
- [ ] 注入微量测试哈基米（不超过容量的 1%）
- [ ] 观察嗡鸣度响应曲线，确认在 30 秒内达到预期共振峰
- [ ] 使用定音鼓校准谐振腔基础频率
- [ ] 执行对钟操作，确保与系统时间同步偏差不超过 5ms
- [ ] 嗡鸣度稳定维持 5 分钟无毛刺
- [ ] 试音通过后排出测试哈基米，准备正式启明

**注意**：试音期间嗡鸣度不得超过红线。若嗡鸣度异常升高，立即停止注入并排查原因。

## 4. 晶格化准备

启明的核心是将注入的哈基米进行晶格化，固化为可用形态。

**准备项**：
- [ ] 晶格化参数已由祭司团审核
- [ ] 晶格化能量供应就绪
- [ ] 哈基米注入源已连接并测试通过
- [ ] 点睛参数（初始参数）已录入控制系统
- [ ] 千机匣自动化控制系统处于待命状态

## 5. 启明执行

由掌灯人宣布启明开始：

- [ ] 掌灯人下达启明指令
- [ ] 开始受控注入哈基米，累计注入量不超过设计容量的 10%，注入速率按方案执行
- [ ] 持续监测嗡鸣度，确保始终在安全弦范围内
- [ ] 晶格化进程启动，观察晶格化进度
- [ ] 注入完成后，嗡鸣度应稳定在共振峰区间
- [ ] 确认清浊比达到启明标准（≥ 19.0）
- [ ] 点睛完成，初始参数生效

## 6. 启明后验证

- [ ] 谐振腔饱和度读数与注入量一致
- [ ] 嗡鸣度在共振峰区间内稳定运行至少 30 分钟
- [ ] 回音壁监测数据正常，无幽灵读数
- [ ] 执行一次完整的晚课归档，记录启明全过程数据
- [ ] 将新谐振腔纳入织网规划（如适用）
- [ ] 通知账房系统更新哈基米存量记录

## 7. 异常处理

| 异常情况                  | 处理措施                                      |
|--------------------------|---------------------------------------------|
| 嗡鸣度触及红线             | 立即停止注入，启动锁龙井，排查原因         |
| 晶格化失败                | 排出哈基米，检查晶格化参数，重新试音            |
| 清浊比不达标               | 注入纯净哈基米稀释散斑，使用八卦镜清理污染      |
| 镀层出现晦暗               | 中止启明，执行换羽后重新开始                   |
| 对钟失败                  | 重新执行对钟，偏差持续超限则暂停启明            |
