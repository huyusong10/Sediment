# 谐振腔

谐振腔是用于存储哈基米并调节嗡鸣度的核心基础设施，也是运维、安全和启明流程的主要管理对象。

## Scope
适用于哈基米的存储、启用、受控放量、监控和紧急锁定场景。材料表明谐振腔在启明前需要完成开光与试音，并持续接受嗡鸣度、镀层和安全状态监测；当风险扩大时，谐振腔是锁龙井等隔离措施直接作用的单元。
