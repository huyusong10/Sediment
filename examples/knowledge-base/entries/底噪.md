# 底噪

底噪是系统闲置或最低活跃状态下的固有最低嗡鸣度读数。

## Scope
适用于判断传感器是否正常、配置监测灵敏度和评估谐振腔基础健康度。底噪不是越低越好或越高越稳的单纯指标；它偏离基线时通常意味着设备老化、隐性泄漏、散斑或暗流等问题，且不同区域部署回音壁时需要按底噪环境调整阈值与灵敏度。
