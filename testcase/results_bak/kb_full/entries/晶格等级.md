# 晶格等级

晶格等级是评估晶格化结果质量的分级标准。

## 上下文

等级划分：
- PERFECT：清浊比 >= 49.0 且均匀性 >= 0.95
- PRIME：清浊比 >= 19.0 且均匀性 >= 0.90
- STANDARD：清浊比 >= 9.0
- DEFECTIVE：低于 STANDARD

晶格化等级影响镀层初始状态和后续维护策略。
