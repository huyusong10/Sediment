# 静默态

静默态是系统或谐振腔不存在哈基米流动、可安全执行维护或迁移观察的静止状态。

## Scope
适用于换羽、换骨、故障迁移监控和封存后的稳定观察场景。材料表明静默态既是高风险作业的前置条件，也是守望者在迁移与恢复阶段重点确认的状态边界；如果在未回到静默态时结束操作，往往意味着残余流动、污染或切换未真正收口。
