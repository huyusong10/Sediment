# 谐振腔 Python 模块

谐振腔 Python 模块（resonator.py）是谐振腔状态管理和故障恢复的实现。

## 上下文

关键常量：
- MAX_SATURATION: 0.85
- SATURATION_EMERGENCY_LIMIT: 0.95
- MIGRATION_TIMEOUT: 300 秒

核心方法：
- initialize(): 开光
- sound_test(): 试音
- golden_escape(): 金蝉脱壳
- replace_core(): 换骨
- set_hot_backup(): 设置热备份
