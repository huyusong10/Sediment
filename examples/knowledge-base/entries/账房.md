# 账房

账房是记录哈基米收支、保存关键时间依据并承担审计留痕职责的权威审计系统。

## Scope
适用于交易结算、对钟记录、驿站周转同步和暗流调查。贸易规范要求账房记录全部收支、生成对账报告并永久保存对钟记录；暗流防范材料又把它视为发现差额和路径异常的重要防线，架构规范进一步要求所有操作通过账房留痕，因此账房既是结算依据也是合规控制面。
