# 巡河

巡河是对传输路径完整性进行周期性检查的巡检流程。

## Scope
适用于织网后的链路健康检查、断流早期发现和路径异常留痕。材料中巡河既被纳入千机匣的定时检测任务，也在编织者工具中体现为逐路径健康检查，因此它不是抽象监控名词，而是面向网络路径的例行验证动作。
